Supports 1-3 arguments, 1 argument would be the number of the condition, which would apply to yourself, and by default and infinite duration, 3 arguments is to apply it to someone else, and the third argument is duration.

Aimbot

12/21/2014

0 Comments

 
More formally known as Auto Aimer, I created it without even looking to see if such a thing existed already, and it was based on an auto reflect plugin that had the option to track players. Features include an aimkey and soon to be FoV aiming. It has a check to see if you can actually see the target which isn't always accurate, and it tends to aim in front of the target for some unknown reason, but it's 100% accurate to the head, you just need to have the target not move perpendicular to you. !aim with no arguments will toggle it on yourself
